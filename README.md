# Referencia Bíblica

Proyecto SPA para lectura de la Biblia en rumano — fetch desde `bible-api.com`.

Cómo ejecutar localmente:

```powershell
# servir archivos estáticos en el puerto 8000
npx http-server -p 8000 .
# abrir http://localhost:8000
```

Notas:
- Contiene componentes en `components/`.
- Usa `script.js` como lógica principal.
